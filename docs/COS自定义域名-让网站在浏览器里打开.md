# COS 自定义域名：让网站在浏览器里正常打开

2024 年后，腾讯云 COS 的**默认域名**（如 `xxx.cos-website.ap-guangzhou.myqcloud.com`）在浏览器里会**触发下载**而不是打开网页。要正常访问，需要给存储桶**绑定自定义域名**。

---

## 一、先有一个域名

- **已有域名**：用你的域名或子域名即可（例如 `profit.你的域名.com`）。
- **没有域名**：在 [腾讯云域名注册](https://dnspod.cloud.tencent.com/) 或 [阿里云](https://wanwang.aliyun.com/) 等注册一个（几十元/年），或使用免费域名服务。

下面以「你已有一个域名」为例。

---

## 二、在 COS 里添加自定义域名

1. 登录 [腾讯云 COS 控制台](https://console.cloud.tencent.com/cos) → 进入你的桶 **bauchi-profit-1411527310**。
2. 左侧点 **域名与传输管理**（在「安全管理」或「基础配置」附近）→ 找到 **自定义源站域名** 或 **自定义域名**。
3. 点 **添加域名** 或 **添加**：
   - **域名**：填你要用来访问网站的地址，例如：
     - `profit.你的域名.com`，或
     - `bauchi.你的域名.com`
     （不要带 `http://` 或 `https://`，只填域名本身。）
   - **源站类型**：选 **静态网站**（不要选默认的「静态网站源站」以外的其他类型，若有「静态网站」就选它）。
   - 若有「回源鉴权」等选项，保持默认即可。
4. 保存后，页面上会显示 **CNAME 目标**（或「记录值」），形如：  
   `bauchi-profit-1411527310.cos-website.ap-guangzhou.myqcloud.com`  
   **复制保存**这个 CNAME 目标，下一步要用。

---

## 三、在 DNS 里添加 CNAME 解析

到你**管理这个域名 DNS 的地方**（腾讯云 DNSPod、阿里云解析、Cloudflare 等）添加一条 **CNAME 记录**：

1. **主机记录**（或「名称」）：
   - 若你要用 `profit.你的域名.com`，就填 **`profit`**；
   - 若你要用 `bauchi.你的域名.com`，就填 **`bauchi`**；
   - 若你要用根域名 `你的域名.com`，就填 **`@`**（部分平台根域名要配 A 记录，具体看平台说明）。
2. **记录类型**：选 **CNAME**。
3. **记录值**（或「目标」）：粘贴上一步在 COS 里复制的 **CNAME 目标**（即 `xxx.cos-website.ap-guangzhou.myqcloud.com`）。
4. **TTL**：默认或 600 即可。
5. 保存。

**腾讯云 DNSPod**： [DNSPod 控制台](https://console.dnspod.cn/) → 选域名 → 添加记录 → 类型 CNAME，主机记录填子域名，记录值填 COS 的 CNAME 目标。

---

## 四、等解析生效后访问

- DNS 生效一般要 **几分钟到几十分钟**。
- 在浏览器地址栏输入：**`https://你绑定的域名`**（例如 `https://profit.你的域名.com`）。
- 若提示证书不受信任，可暂时用 **`http://`** 访问（部分环境需在 COS 或 CDN 里开启 HTTPS，见下）。

---

## 五、（可选）开启 HTTPS

若希望用 `https://` 且浏览器不报错：

- 在 **域名与传输管理** 里，找到你刚添加的自定义域名，看是否有 **SSL 证书** 或 **HTTPS 配置**。
- 腾讯云 COS 自定义域名通常可**免费申请/绑定一年 SSL 证书**，按页面提示申请并绑定到该域名即可。
- 绑定成功后，用 `https://你的域名` 访问即可。

---

## 六、小结

| 步骤 | 做什么 |
|------|--------|
| 1 | 有一个可管理的域名 |
| 2 | COS → 域名与传输管理 → 添加自定义域名（填域名、选静态网站）→ 复制 CNAME 目标 |
| 3 | 到 DNS 处添加 CNAME 记录：主机记录 = 子域名，记录值 = CNAME 目标 |
| 4 | 等几分钟后用 `https://你的域名` 或 `http://你的域名` 打开网站 |

这样网站就会在浏览器里正常打开，而不会再变成下载。
